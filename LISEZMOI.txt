PRESENTATION :
==============
BelgianEidAuthPlugin est un plugin d'authentification pour PAS (PluggableAuthService).  Ce plugin permet à un utilisateur de s'identifier à l'aide de sa carte d'identité électronique belge.

L'identification classique par nom d'utilisateur et mot de passe fonctionne toujours.

INSTALLTION :
=============
Attention, ce plugin nécessite PluggableAuthService version 1.2 (non testé avec une version plus ancienne).  Testé avec Plone 2.5rc2, Plone 2.5rc3.

--> Copier le produit dans le répertoire 'Products' d'une instance Zope
--> Dans la ZMi, aller dans acl_users et ajouter un BelgianEidCardPlugin.
--> Activer le plugin dans plugins/Authentication Plugins

Vous devez également configurer un serveur Web (Apache, IIS) pour que celui-ci accepte HTTPS et les certificats SSL.
Rendez-vous sur www.communesplone.be pour plus de détails.
